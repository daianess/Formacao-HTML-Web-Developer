Estrutura básica do HTML

O arquivo HTML possui 3 Tags básicas que são essenciais para a identificação do arquivo pelo navegador. São elas: html, head e body.

  • html:
  • head:
  • body: é o corpo do documento, ou seja, nele está contído tudo que o usuário verá na tela
Essas Tags ficam dentro de < > e para realizar seu fechamentoé necessário usar /. Por exemplo < (nome da tag) > para abrir e < / (nome da tag) > para fechar.
